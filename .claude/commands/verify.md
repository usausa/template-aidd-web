allowed-tools: Bash(dotnet:*)

コード単位が固まったら実行し、結果を表で示す。赤があれば直してから次へ。

1. **ビルド(警告0)**  →  !`dotnet build`
2. **テスト**(単体 + 結合。Web は `WebApplicationFactory`)  →  !`dotnet test`

- 失敗・警告があれば原因と対象を列挙し、**緑になるまでその場で直す**(自己修正ループ)。
- 実装途中の未完コードでは失敗して当然。**単位が固まった時**に打つ(編集ごとには回さない = ノイズ回避)。
- `/done` はこの build + test 緑を完了条件に含む。
